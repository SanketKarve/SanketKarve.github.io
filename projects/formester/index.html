Formester

Create custom forms with no programming

https://formester.com

March 2017 - Oct 2017 (Product Acquired)

Formester help user to create customized forms for various purposes like blogging, Mass digital marketing, subscription, invites and much more. Formester provides simple drag and drop tool to create the form and you can just add a script to your code. It provides features like drag and drop, Mail List, Auto Responder, interactive dashboard, Spam Protection and Integrating with various applications like slack, Gmail, and SalesForce.
