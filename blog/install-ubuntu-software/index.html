Things Developer Should Do After Installing Ubuntu

List of software and packages developer should install after fresh Ubuntu installation

August 5, 2017 Sanketkumar Karve

3 minute read

  • Twitter

  • Google+

  • LinkedIn

  • Email

You have freshly install Ubuntu on you system and thinking that there are lot of software and packages you want to install to get back to work ASAP. Here is quick checklist for the Developer for all software and packages you will need.  

1. Update and Upgrade

On fresh installation many software and packages does not provide latest updates, which you can obtain by using sudo apt-get update command. If any new upgrade for Ubuntu is there sudo apt-get upgrade will help to install it.

sudo apt-get update && sudo apt-get upgrade

2.Enable Partner Repositories

The partner repositories are another source of software, but for those with restrictive licenses that mean the general public cannot simply share them, such as Flash and Skype. These are official repo’s that are already added but disabled in Ubuntu. This command enables them.

3.Synaptic Package Manager

Remember our major task is to install lots of software and packages. Synaptic is a graphical package management tool based on GTK+ and APT. Synaptic enables us to install, upgrade and remove software packages in a friendly way.

sudo apt-get install synaptic

4.Google Chrome

Ubuntu comes with Chromium browser by default.Chromium is the open-source project that forms the basis for Google Chrome. Chrome is commercial and has number of proprietary, closed-source bits that Chromium lacks like AAC, H.264, and MP3 Support,Adobe Flash (PPAPI),Google Update,Extension Restrictions,Crash and Error Reporting and Security Sandbox. You can download Chrome latest file from this link. or you can use following command :

wget -q -O - https://dl-ssl.google.com/linux/linux_signing_key.pub | sudo apt-key add -
sudo sh -c 'echo "deb http://dl.google.com/linux/chrome/deb/ stable main" >> /etc/apt/sources.list.d/google.list'
sudo apt-get update
sudo apt-get install google-chrome-stable

Here are few list of chrome extension you may required a lot:

  • Add Blocker or Add Blocker Plus.
  • Grammarly.
  • HTML CSS Live Editor.
  • Postman Rest Client.
  • Web server for chrome.
  • Media Downloader.
  • Social media extension (Gmail, Hangout, Facebook, Twitter, etc)

5.Java

Java is need for running many application’s on Linux platform. First check if Java is installed or not(Usually it open JDK by default).

java -version

If not installed you can use following command to install Java.

sudo apt-get install default-jdk

For Installing Oracle JDK(for version 8) you can use.

sudo apt-get install python-software-properties
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java8-installer

Managing Java(Optional).

sudo update-alternatives --config java

Now select default Java type.

5.Python

If you work on Python 3.5 then you can following command:

sudo apt-get install build-essential checkinstall
sudo apt-get install libreadline-gplv2-dev libncursesw5-dev libssl-dev libsqlite3-dev tk-dev libgdbm-dev libc6-dev libbz2-dev
cd /usr/src
sudo wget https://www.python.org/ftp/python/3.5.2/Python-3.5.2.tgz
cd Python-3.5.2
sudo ./configure
sudo make altinstall
python3.5 -V

6.Text Editor

Here are popular text editor you can download:

  • Atom
sudo add-apt-repository ppa:webupd8team/atom
sudo apt-get update
sudo apt-get install atom
  • Sublime text editor
sudo add-apt-repository ppa:webupd8team/sublime-text-3
sudo apt-get update
sudo apt-get install sublime-text-installer

7. Git

Git is popular distributed version control system. Many projects maintain their files in a git repository, and sites like GitHub and Bitbucket have made sharing and contributing to code simple and valuable.

sudo apt-get update
sudo apt-get install git

This are few things that you can install.
